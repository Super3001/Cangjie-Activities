## 冯新宇教授：介绍新一代编程语言——仓颉

1. 仓颉语言的定位：

如同Java, Go, Swift, 仓颉是一门重业务开发语言，具有静态类型和动态内存管理，并且性能优于前述语言。

函数式、命令式、面向对象多范式编程。

2. 相较于其他重业务开发语言，仓颉语言有着丰富的核心语言特性

|                 | Java |  Go  | Swift | Kotlin |  Cangjie  |

|     值类型      |  N   |  Y   |   Y   |   N    |     Y     |
|       GC        |  Y   |  Y   |   N   |   Y    |     Y     |
|   用户态线程    |  Y   |  Y   |   -   |   -    |     Y     |
